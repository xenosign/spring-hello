# 단축키

- 변수 자동 생성 : ctrl + alt + V
- 테스트 생성 : ctrl + shift + T
- 스태틱 임포트 생성 : alt(option) + enter 
- 메소드 추출 : ctrl + alt + M
- 배열 자동 반복문 생성 : iter
- 클래스 전체 검색 :윈도우 ctrl + F / 맥 cmd + O
